# ZenChat - Minimalist Chat Interface

A beautiful, minimalist chat interface with glassmorphism design. This is a frontend-only UI demonstration with mock responses.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm run dev
   ```

The app will run on `http://localhost:3000`
